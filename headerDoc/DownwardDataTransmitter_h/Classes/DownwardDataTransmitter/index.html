downwardRequestWithJsonTask:JsonIdentity:

下行同步申请 

- (NSString *) downwardRequestWithJsonTask: (NSString *)jsonTask 
        JsonIdentity: (NSString *)jsonIdentity;
Parameters
jsonTask

JSON字符串结构的任务信息，包括任务标识、应用代码、辅助标记、任务名称、同步条件、创建时间、任务来源、数据包尺寸等内容。

jsonIdentity

经讨论，修改为类Session形式的校验值。//JSON字符串结构的用户身份信息，如：用户名、密码等等，根据应用需要来定义。

Return Value

返回下行同步令牌，其内容实际上就是在服务端针对客户端任务信息而所建任务的标识。

downwardTransmitWithToken:Offset:Length:

下行文件传输 

- (NSString *) downwardTransmitWithToken: (NSString *)token 
        Offset: (long)offset Length: (long)length;
Parameters
token

由DownwardRequest方法返回的下行同步令牌。

offset

文件偏移量，指明本次读取的数据在文件中的起始位置。

length

数据长度，指明要读取的字节数。

Return Value

返回本次接收的文件数据。
