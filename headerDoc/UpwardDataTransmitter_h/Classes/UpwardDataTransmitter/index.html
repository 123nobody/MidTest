upwardFinishWithToken:Trash:

上行文件传输结束(一个文件) 

- (NSString *) upwardFinishWithToken: (NSString *)token 
        Trash: (NSString *)trash;
Parameters
token

由UpwardRequest方法返回的上行同步令牌。

trash

指示是否废弃当前任务。false标识正常结束；true标识非正常结束，如客户端在传输任务过程中，用户要求删除该任务或者撤销任务，就需要向服务端发送一条废弃当前任务的结束命令。

Return Value

成功返回YES，失败返回NO。

upwardRequestWithJsonTask:JsonIdentity:

上行同步申请 

- (NSString *) upwardRequestWithJsonTask: (NSString *)jsonTask 
        JsonIdentity: (NSString *)jsonIdentity;
Parameters
jsonTask

JSON字符串结构的任务信息，包括任务标识、应用代码、辅助标记、任务名称、同步条件、创建时间、任务来源、数据包尺寸等内容。

jsonIdentity

经讨论，修改为类Session形式的校验值。//JSON字符串结构的用户身份信息，如：用户名、密码等等，根据应用需要来定义。

Return Value

返回上行同步令牌，其内容实际上就是在服务端针对客户端任务信息而所建任务的标识。

upwardTransmitWithToken:FileName:Offset:Base64String:

上行文件传输 

- (NSString *) upwardTransmitWithToken: (NSString *)token 
        FileName: (NSString *)fileName Offset: (long)offset 
        Base64String: (NSString *)base64String;
Parameters
token

由UpwardRequest方法返回的上行同步令牌。

fileName

本次上传的文件名。

offset

文件偏移量，指明本次发送的数据在文件中的起始位置。

base64String

base64编码后的数据，数据缓冲区，存储文件段数据,要求服务端把此数据写入文件中，写入的起始位置是lOffset。

Return Value

成功返回YES，失败返回NO。
